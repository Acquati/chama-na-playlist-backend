# ChamaNaPlaylist - Backend

Steps to run this project in development:

1. Run `cp .env.example .env` command
2. Run `yarn install` command
3. Run `docker-compose up` command
4. Run `yarn start` command

Test:

1. Run `cp .env.example .env` command
2. Run `yarn install` command
3. Run `docker-compose up` command
4. Run `yarn test` command
